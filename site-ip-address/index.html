Overview

A site IP address is a numerical label assigned to each machine connected to a computer system that uses the Internet Protocol for communication. It enables devices to locate and interact with each other over the internet. There are two primary versions of IP addresses: IPv4 and IPv6, each serving the same fundamental purpose but with distinct characteristics.

How It Works

When you type a website URL into your browser, your device queries a Domain Name System (DNS) server to translate the domain name into its corresponding site IP address. This IP address directs your request to the correct server hosting the website, allowing the information to be sent back to your device.

IPv4 vs IPv6

IPv4 (Internet Protocol version 4) is the most widely used IP addressing system, consisting of a 32-bit number typically represented in decimal format (e.g., 192.168.1.1). However, due to the limited number of unique addresses available in IPv4, the world has gradually transitioned to IPv6. IPv6 (Internet Protocol version 6) uses a 128-bit address, allowing for an almost infinite number of unique IP addresses, which is essential for accommodating the growing number of devices connected to the internet.
